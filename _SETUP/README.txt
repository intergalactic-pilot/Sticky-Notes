NoteBot - Modern UI Edition - Setup Files
==========================================

This folder contains all the files used to build the Windows installer and the .deb package.

IMPORTANT: This is the Modern UI Edition fork of NoteBot
---------------------------------------------------------
This enhanced version includes:
- Windows 11 Fluent Design UI with rounded corners
- 9 modern color schemes including dark theme
- Enhanced resize experience with visual feedback
- Modern hover effects and animations
- Improved usability and DPI scaling

For the original NoteBot project by Federico Dossena, visit:
http://notebot.fdossena.com/

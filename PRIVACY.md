# Privacy Policy

## NoteBot Modern UI Edition

### Data Collection

**NoteBot Modern UI Edition collects NO personal data.** This application is completely offline and local.

### What We Don't Do

❌ **No telemetry** - We do not send any usage data  
❌ **No tracking** - We do not track your activity  
❌ **No cloud sync** - All notes stay on your computer  
❌ **No ads** - No advertisements or analytics  
❌ **No accounts** - No login or registration required  
❌ **No data sharing** - We never share any information  

### Local Storage Only

All your notes are stored locally on your computer at:
- **Windows:** `%APPDATA%\NoteBot\`

You have full control over your data:
- You can back up your notes anytime
- You can delete your notes anytime
- You own your notes - nobody else has access

### Source Code Transparency

This is an **open-source project** under GPL-3.0 license. You can:
- Review all source code on GitHub
- Verify there are no hidden behaviors
- Modify and compile the application yourself
- Distribute modified versions (respecting GPL-3.0)

### Java Runtime

NoteBot Modern UI Edition requires Java Runtime Environment (JRE). Java's privacy policies are separate from this application:
- Oracle Java Privacy Policy: https://www.oracle.com/legal/privacy/
- OpenJDK: Community-driven, no data collection

### Third-Party Services

This application does not integrate with any third-party services.

### Changes to This Policy

Since NoteBot is open-source, any changes to privacy practices would be visible in the source code commits. Updates to this document will be announced in the release notes.

### Questions?

If you have privacy concerns:
1. Review the open-source code on GitHub
2. Compile it yourself to verify behavior
3. Open an issue on GitHub for clarification
